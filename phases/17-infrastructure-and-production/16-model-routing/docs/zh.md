# 模型路由作为成本削减的基石

> 动态代理会评估每个请求（任务类型、token长度、嵌入相似度、置信度），将简单查询发送给廉价模型，将复杂问题升级到前沿模型。也被称为模型级联。生产案例研究表明，在美国/英国/欧盟的部署中，在保持相同质量的前提下，成本降低了20-60%；在高流量SaaS平台上，路由效率提升30%可转化为六位数的年度节省。2026年的背景是，LLM推理价格每年下降约10倍——从2022年末到2026年，一个GPT-4级别的token价格从20美元/百万降至约0.40美元/百万。大部分降幅来自更高效的服务技术栈（第17阶段 · 04-09），而非硬件。路由正是你如何在产品不倒退的情况下，将这种价格下降转化为利润的方式。其失败模式是廉价模型偏移：路由将40%的请求推送到较弱的模型，导致推理任务质量下降3-5%，但一个季度内都无人察觉。需通过在线质量指标（而非仅离线评估集）来把控路由。

**先修内容：** 第17阶段 · 01（托管LLM平台），第17阶段 · 19（AI网关）

## 学习目标

- 解释模型级联：廉价优先，辅以置信度检查，低置信度时升级。
- 列举四种路由信号（任务分类、提示长度、与已知难题集的嵌入相似度、初次运行的自我置信度）。
- 在给定路由分配比例和质量损失容忍度下，计算预期的混合成本。
- 说出捕获廉价模型性能滑坡的偏移监控指标（在线质量门控）。

## 问题所在

你的服务每月在GPT-5上花费8万美元。分析显示，70%的查询是简单的：“巴黎现在几点？”“改写这个句子。”一个Haiku级别的模型能完美处理这些，成本仅为3%。另外30%需要GPT-5的推理能力——编码、数学、多步骤规划。

如果你将70%路由到廉价模型，30%路由到昂贵模型，在相同产品质量下，你的账单将降低约65%。这就是路由。诀窍是在构建代理时不降低质量。

## 核心概念

### 四种路由信号

1.  **任务分类**：简单/复杂/代码生成/数学/聊天。可以是基于规则的分类器、小型LLM（成本约0.25美元/百万的Haiku级别模型），或与标注桶的嵌入相似度。输出：路由 = 廉价 / 均衡 / 前沿。

2.  **提示长度**：超过4K token的提示通常需要前沿模型以确保连贯性。低于500 token的提示通常不需要。

3.  **与已知难题集的嵌入相似度**：如果查询与已知难题桶接近（余弦相似度 > 0.88），则直接升级到前沿模型。

4.  **初次运行的自我置信度**：先发送给廉价模型；如果模型的日志概率显示低置信度、或拒绝回答、或输出含糊语言，则在前沿模型上重试。这会给约10%的流量增加P95延迟，但能为其他90%的流量节省50%以上的成本。

### 三种模式

**前置路由**（分类器在前）：增加约5-10毫秒延迟；整体最快。

**级联**（廉价优先，低置信度升级）：中位延迟约1.2倍（廉价运行加验证），升级时约2倍。质量下限最高。

**集成路由**（对样本并行运行廉价和前沿模型，奖励模型选择）：质量最高，成本最高；仅用于关键的A/B测试。

### 实现方式

AI网关（第17阶段 · 19）提供了路由功能。LiteLLM具有`router`配置，支持回退和成本路由。Portkey提供防护和路由功能。Kong AI网关提供基于插件的路由。OpenRouter的模型市场提供了推荐API。

开源方案：RouteLLM（LMSYS）、Not Diamond（商业）、Prompt Mule。

### 2026年价格曲线

| 模型类别 | 2022年末 | 2026年 | 变化 |
|-------------|-----------|------|--------|
| GPT-4级别质量 | 约20美元/百万 | 约0.40美元/百万 | 便宜50倍 |
| 前沿（GPT-5, Claude 4） | — | 约3-10美元/百万 | 新的层级 |

大部分改进来自服务效率——第17阶段 · 04-09的核心经验转化成了服务提供商的成本下降。路由让你能在应用层捕获这些收益，而无需等待所有用户迁移到廉价层级。

### 偏移是真正的风险

你的路由将40%的请求发送给廉价模型。在六个月内，任务分布发生变化（用户变得更熟练，问题更长）。路由器没有察觉，因为其分类器是基于第一季度的数据训练的。质量悄然下降。没有人强烈抱怨。你在竞争对手的基准测试中才发现自己落后了。

需通过在线质量指标来把控路由：

- 每条路由的用户点赞/点踩。
- 对每条路由预留样本（5%）进行自动化LLM判断。
- 升级率：如果级联的上路由比例超过30%，说明廉价模型被过度路由。
- 每条路由的拒绝率。

### 你应该记住的数字

- 2026年在相同质量下通过路由节省的成本：案例研究显示为20-60%。
- LLM价格在2022-2026年的下降：总计约每年10倍。
- GPT-4级别 2022年 vs 2026年：约20美元/百万 → 约0.40美元/百万。
- 级联对延迟的影响：中位约1.2倍，升级时约2倍（约10%的流量）。

## 关键术语

| 术语 | 人们怎么说 | 实际含义 |
|------|----------------|------------------------|
| 模型路由 | “成本代理” | 为每个请求动态选择模型 |
| 模型级联 | “廉价优先，后升级” | 运行廉价模型，低置信度时回退到前沿模型 |
| 前置路由 | “先分类” | 分类器在前；无需重新运行 |
| 集成路由 | “并行选择” | 运行多个模型，奖励模型选择最佳 |
| 升级率 | “上路由百分比” | 级联请求中被升级的比例 |
| RouteLLM | “LMSYS路由器” | 开源路由器库 |
| Not Diamond | “商业路由器” | SaaS模型路由产品 |
| 偏移 | “廉价滑坡” | 分布变化但路由器未察觉 |
| 在线质量门控 | “实时检查” | 自动化LLM判断实时流量采样 |
