# 生成模型 — 分类与历史

> 每一个图像模型、文本模型、视频模型和3D模型都归属于这五大类别之一。选错了类别，你可能会与数学搏斗数周；选对了，这个领域过去十二年的进展就能在你的脑海中清晰地堆叠成型。

**类型:** 学习
**语言:** Python
**前置要求:** 第二阶段（机器学习基础）、第三阶段（深度学习核心）、第七阶段 · 14（Transformer）
**时间:** ~45分钟

## 问题

生成模型只做一件事：给定从未知分布`p_data(x)`中抽取的训练样本，输出看起来像是来自同一分布的新样本。人脸、句子、MIDI文件、蛋白质结构——如果你眯起眼睛看，都是同一个问题。

难点在于`p_data`存在于一个拥有数百万维度的空间中（一张512x512的RGB图像约有78.6万个维度），样本位于该空间内部一个薄薄的流形上，而你可能只有大约1000万个样本。暴力计算密度是无望的。每个生成模型都是一种妥协，用一个略次难的难题换取一个更难的难题。

过去十二年存活下来的是五个家族。了解每个家族做了什么妥协，就能明白它为何在某些任务上获胜，而在其他任务上崩溃。

## 概念

![生成模型的五个家族——按它们建模的内容分类](../assets/taxonomy.svg)

**1. 显式密度，可解。** 将`log p(x)`写成一个你实际可以计算的和。自回归模型（PixelCNN、WaveNet、GPT）将`p(x) = ∏ p(x_i | x_<i)`因式分解。归一化流（RealNVP、Glow）将`p(x)`构建为简单基础分布的可逆变换。优点：精确的似然，干净的训练损失。缺点：自回归推理是顺序的（对长序列很慢），流需要可逆的架构（架构限制性）。

**2. 显式密度，近似。** 将`log p(x)`的下界（ELBO）进行优化。VAE（Kingma 2013）使用编码器-解码器结构和变分后验。扩散模型（DDPM、Ho 2020）训练一个去噪器，隐式地优化加权ELBO。扩散是2026年主导的图像、视频和3D骨干模型。

**3. 隐式密度。** 完全跳过密度；学习一个生成器`G(z)`来产生样本，以及一个判别器`D(x)`来区分真假。GAN（Goodfellow 2014）。推理速度快（一次前向传播），但训练过程极其不稳定。直到2026年，StyleGAN 1/2/3在固定领域的照片级写实（人脸、卧室）方面仍是顶尖水平。

**4. 基于分数/连续时间。** 直接学习对数密度`∇_x log p(x)`的梯度（分数）。Song & Ermon (2019) 表明分数匹配将扩散推广到了SDE。流匹配（Lipman 2023）是2024-2026年的热门：无需模拟的训练、更直的路径、比DDPM快4-10倍的采样。Stable Diffusion 3、Flux、AudioCraft 2都使用流匹配。

**5. 基于离散码的自回归。** 使用VQ-VAE或残差量化器将高维数据压缩成一个短离散token序列，然后用Transformer建模该token序列。Parti、MuseNet、AudioLM、VALL-E、Sora的patch分词器都使用这种方法。这是第一类加上一个学习到的分词器。

## 五问分流法

当一篇新的生成模型论文出来时，在阅读方法部分之前，先回答这五个问题。

1.  **建模对象是什么？** 像素、潜变量、离散token、3D高斯、网格、波形？
2.  **密度是显式还是隐式？** 他们是否写下了`log p(x)`？
3.  **采样：一次性还是迭代？** 迭代意味着推理更慢；一次性通常意味着对抗性或蒸馏。
4.  **条件化：无条件、类、文本、图像、姿态？** 这决定了损失和架构框架。
5.  **评估指标：FID、CLIP分数、IS、人类偏好、任务准确率？** 每个指标都有已知的失效模式（见第14课）。

在这个阶段的每一课中，你都将重新回答这五个问题。最后，它们将成为你的本能反应。

## 动手实现

本课的代码是一个轻量级可视化：用三种玩具方法（核密度、离散直方图和一个最近样本"类GAN"生成器）从样本拟合一维混合高斯分布，这样你就能在单屏打印的问题上看到显式密度与隐式密度的区别。

运行`code/main.py`。它从一个双峰高斯混合分布中抽取2000个样本，然后打印：

```
explicit density (histogram): p(x in [-0.5, 0.5]) ≈ 0.38
approximate density (KDE):     p(x in [-0.5, 0.5]) ≈ 0.41
implicit (nearest-sample gen): 20 new samples printed, no p(x)
```

注意：前两个方法让你可以问"这个点的概率是多少？"。第三个则不能。这就是*显式与隐式*的区别，它将在未来的每一课中都很重要。

## 应用

在2026年，哪个家族适合哪个任务？

| 任务 | 最佳家族 | 原因 |
|------|-------------|-----|
| 照片级写实人脸，狭窄领域 | StyleGAN 2/3 | 仍最清晰、推理最快。 |
| 通用文生图 | 潜空间扩散 + 流匹配 | SD3, Flux.1, DALL-E 3. |
| 快速文生图 | 修正流 + 蒸馏 | SDXL-Turbo, SD3-Turbo, LCM. |
| 文生视频 | 扩散 Transformer + 流匹配 | Sora, Veo 2, Kling. |
| 语音 + 音乐 | 基于token的AR（AudioLM, VALL-E, MusicGen）或流匹配（AudioCraft 2） | 离散token扩展成本低。 |
| 3D场景 | 高斯泼溅拟合、扩散先验 | 3D-GS用于重建，扩散用于新视角。 |
| 密度估计（无采样） | 流 | 唯一具有精确`log p(x)`的家族。 |
| 模拟 / 物理学 | 流匹配、分数SDE | 直线路径、平滑向量场。 |

## 关键术语

| 术语 | 常见说法 | 实际含义 |
|------|-----------------|-----------------------|
| 生成模型 | "它能创造新东西" | 学习一个`p_data(x)`的采样器，可选暴露`log p(x)`。 |
| 显式密度 | "你可以计算它" | 模型提供闭式或可解的`log p(x)`。 |
| 隐式密度 | "GAN风格" | 仅是一个采样器——无法评估给定点的`p(x)`。 |
| ELBO | "证据下界" | `log p(x)`的一个可解下界；VAE和扩散模型优化它。 |
| 分数 | "对数密度的梯度" | `∇_x log p(x)`；扩散和SDE模型学习这个场。 |
| 流形假设 | "数据在一个表面上" | 高维数据集中在一个低维流形上；这是降维有效的原理。 |
| 自回归 | "预测下一部分" | 将联合分布因式分解为条件分布的乘积。 |
| 潜变量 | "压缩编码" | 低维表示，解码器可以从中重建输入。 |

## 生产备忘：五个家族，五种推理形态

每个家族对应不同的推理服务器成本曲线。生产推理文献将LLM推理视为预填充+解码；这里的分解也适用：

- **自回归（第1和第5类）。** 顺序解码主导延迟；KV缓存、连续批处理和推测解码都直接适用。
- **VAE / 扩散 / 流匹配（第2和第4类）。** 没有LLM意义上的解码。成本 = `num_steps × step_cost`，而`step_cost`是在全潜空间分辨率下的Transformer或U-Net前向传播。生产中的调节旋钮是步数（DDIM / DPM-Solver / 蒸馏）、批大小和精度（bf16 / fp8 / int4）。
- **GAN（第3类）。** 一次前向传播。没有调度，没有KV缓存。TTFT ≈ 总延迟。这就是为什么StyleGAN在狭窄领域的用户体验方面仍然获胜。

当你在论文摘要中看到"比扩散更快"时，将其翻译为"更少的步骤 × 相同的步骤成本"或"相同的步骤 × 更低的步骤成本"。其他都是营销。
