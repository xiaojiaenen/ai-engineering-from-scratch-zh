# 大语言模型的差分隐私

> DP-SGD 仍是标准方案——注入噪声的梯度更新提供形式化的（ε, δ）保证。计算、内存和效用开销显著；参数高效的 DP 微调（LoRA + DP-SGD）是 2025 年的常见配置（ACM 2025）。两组证据存在矛盾：基于金丝雀的成员推断攻击（Duan 等，2024）报告其对语言模型效果有限；训练数据提取（Carlini 等，2021；Nasr 等，2025）则恢复了大量逐字记忆内容。2025 年 3 月的解析（arXiv:2503.06808）：差异在于衡量标准不同——插入的金丝雀 vs "最易提取"的数据。新的金丝雀设计实现了无需影子模型的基于损失的成员推断攻击（MIA），并首次对一个在真实数据上训练、具有现实 DP 保证的大语言模型进行了有意义的 DP 审计。替代方案：PMixED（arXiv:2403.15638）——通过专家混合模型在下一个 token 分布上实现推理时的私有预测；DP 合成数据生成（Google Research 2024）。新兴攻击：通过大语言模型反馈的差分隐私逆转——置信度分数泄露。

## 问题所在

大语言模型会记忆。Carlini 等人 2021 年的研究表明，生产环境中的语言模型能按需逐字复现训练文本。DP 是形式化的防御手段：训练时使输出对任何单个训练样本都具有可证明的不可区分性。2024-2025 年的证据表明 DP-SGD 是必要的，但部署的 ε 值可能与威胁模型不匹配。

## 核心概念

### (ε, δ)-差分隐私

一个随机化算法 M 是 (ε, δ)-DP 的，当且仅当对于任何仅相差一个样本的两个数据集 D 和 D' 以及任何事件 S：
P(M(D) in S) <= e^ε * P(M(D') in S) + δ。

解释：输出分布足够接近（由 ε 参数化），使得任何单个个体的贡献无法被可靠推断，除非以 δ 的概率。

### DP-SGD

Abadi 等人 2016 年提出。标准方案：
1. 采样一个迷你批次。
2. 计算每个样本的梯度。
3. 将每个样本的梯度裁剪到阈值 C。
4. 将裁剪后的梯度求和，并加入标准差为 σ * C 的高斯噪声。
5. 使用带噪的和来更新参数。

隐私开销由会计员（矩会计员、雷尼 DP 会计员）跟踪。大语言模型文献中报告的 ε 值因威胁模型、数据敏感性和效用目标而差异很大；没有普遍 "安全" 的默认 ε。已发布的示例在某些大语言模型训练设置中大约在 ε ≈ 1–10 之间，但这些仅作说明用——并非推荐的默认值。较低的 ε 通常需要更多噪声，并可能增加效用损失。

### LoRA + DP-SGD

对前沿模型进行完全的 DP-SGD 开销巨大。LoRA（Hu 等 2022）将梯度更新限制在一个小的适配器上，减少了每个样本梯度的存储需求。LoRA + DP-SGD 是 2025 年的常见配置。DP 保证适用于适配器；基础模型保持固定。

### 2024-2025 年的矛盾

两组证据：

- **金丝雀 MIA（Duan 等 2024）。** 在训练数据中插入独特的金丝雀样本，测量成员推断攻击者是否能识别它们。报告显示其在语言模型上效果有限。表明 MIA 很困难。
- **训练数据提取（Carlini 2021，Nasr 等 2025）。** 用前缀提示模型；测量其是否能从训练数据中恢复逐字文本。报告显示存在大量记忆。表明在相关意义上 MIA 很容易。

2025 年 3 月的解析（arXiv:2503.06808）：两者衡量的事物不同。MIA 询问 "样本 e 在数据集 D 中吗？"，针对的是插入的金丝雀。提取询问 "我能从 D 中恢复什么？"。对于隐私而言，重要的是 "最易提取" 的样本；金丝雀低估了这一点，因为它们未被优化以利于提取。

新的金丝雀设计。无需影子模型的基于损失的 MIA。首次对一个在真实数据上训练、具有现实 DP 保证的大语言模型进行了有意义的 DP 审计。

### DP 训练的替代方案

- **PMixED (arXiv:2403.15638)。** 推理时的私有预测。基于下一个 token 分布的专家混合模型；每个专家查看训练数据的一个分片；聚合时为 DP 添加噪声。完全避免了 DP 训练。
- **DP 合成数据生成（Google Research 2024）。** 使用 DP-SGD 对 LoRA 进行微调，采样合成数据，在合成数据上训练下游分类器。

两者都以不同的威胁模型为代价，绕过了完全 DP 训练的效用成本。

### 通过大语言模型反馈的差分隐私逆转

2025 年的新兴攻击。使用 DP 训练模型的置信度分数作为预言机来重新识别个体。即使输出本身不泄露，置信度分布也可能泄露信息。

防御措施：不要暴露置信度，或在暴露前进行截断/量化。这是超出 (ε, δ)-DP 训练的额外要求。

## 实践应用

`code/main.py` 在一个玩具二分类数据集上模拟 DP-SGD。你可以扫描噪声乘数 σ 和裁剪范数 C，并跟踪 (ε, δ) 预算和准确率成本。一个 "金丝雀攻击" 插入一个独特的训练样本，并测量在 DP 前后基于对数损失的测试是否能检测到它。

## 关键术语

| 术语 | 常用说法 | 实际含义 |
|------|----------|----------|
| DP | "(ε, δ)-差分隐私" | 形式化隐私：输出分布在邻近数据集变化下保持接近 |
| DP-SGD | "注入噪声的 SGD" | 梯度裁剪 + 高斯噪声添加；标准 DP 训练 |
| LoRA + DP-SGD | "高效私有微调" | 在低秩适配器上进行 DP-SGD；2025 年标准配置 |
| MIA | "成员推断" | 确定样本是否在训练数据中的攻击 |
| 金丝雀 | "插入的水印样本" | 用于衡量 DP 泄露的独特训练样本 |
| PMixED | "私有推理混合" | 通过专家混合模型在下一个 token 分布上实现推理时 DP |
| DP 逆转 | "置信度泄露攻击" | 利用模型置信度作为预言机进行重新识别的攻击 |
