# 心智理论与涌现协调

> Li等人（arXiv:2310.10701）的研究表明，合作文本游戏中的LLM代理表现出**涌现的高阶心智理论**——即推理另一个代理对第三个代理信念的信念——但由于上下文管理和幻觉问题，在长时程规划上表现不佳。Riedl（arXiv:2510.05174）在群体层面测量了高阶协同效应，发现**只有**在心智理论提示条件下才会产生与身份关联的分化和目标导向的互补性；低能力LLM仅表现出虚假的涌现现象。这意味着协调涌现是依赖提示且与模型相关的，并非自然产生。本课程实现一个最小化的心智理论感知代理，在有无心智理论提示下运行合作任务，并根据Riedl 2025协议测量协调差异。

**类型：** 学习 + 构建
**语言：** Python（标准库）
**前置课程：** 第16章 · 07（心智社会与辩论），第16章 · 17（生成式代理）
**时间：** 约75分钟

## 问题

多代理协调常看似神奇：代理分工协作、相互预见、避免冗余。通常这种"涌现"是提示工程的产物——有人告诉代理要"协调"。移除提示，协调便消失。

Riedl 2025年的研究更为严格：在受控条件下，协调只在代理被提示去推理**其他代理的心智**（心智理论）时涌现。若无心智理论提示，即使强大的模型也表现出无法通过统计检验的协调模式。这对生产环境至关重要：团队发布的"多代理协调"功能往往依赖提示且脆弱。

本课程将心智理论视为一种特定能力（对信念的信念进行推理），构建一个最小化的心智理论感知代理，并测量真正的协调与提示修饰的协调之间的区别。

## 概念

### 心智理论的含义

发展心理学：3岁儿童认为任何人的内心世界都与自己相同。5岁儿童理解他人有不同的信念。7岁儿童能推理关于信念的信念（"她认为我认为球在杯子下面"）。这些分别是零阶、一阶和二阶心智理论。

对于LLM代理，心智理论的阶次映射为：

- **零阶：** 没有他人模型。代理仅根据自身观察行动。
- **一阶：** 代理拥有对其他每个代理信念的模型。"爱丽丝相信X。"
- **二阶：** 代理建模递归信念。"爱丽丝相信鲍勃相信X。"

Li等人2023年的研究发现，一阶和二阶心智理论在LLM代理的游戏中会涌现，但会随着长时程和不可靠通信而退化。

### 萨利-安妮测试简介

1985年的错误信念测试：萨利将弹珠放入篮子A后离开。安妮将其移到篮子B。萨利回来后会去哪里寻找？具有一阶心智理论的儿童会说篮子A（萨利的信念与现实不同）。没有心智理论的儿童会说篮子B。

当提问方式简单明了时，GPT-4时代的LLM能通过萨利-安妮式测试。但当叙述冗长、场景多次变化或问题以间接方式提出时，它们会失败。这是2026年生产环境中LLM心智理论的实际状态。

### Riedl的协调测量方法

Riedl（arXiv:2510.05174）构建了一个群体规模测试：N个代理，一个合作目标，可变的提示条件。测量：

1. **与身份关联的分化。** 代理是否随时间发展出稳定的角色区分？
2. **目标导向的互补性。** 代理的行动是否相互补充（不同子任务）而非重复？
3. **高阶协同效应。** 一种统计度量，衡量群体是否实现了任何子集无法实现的效果。

结果：只在心智理论提示条件下，所有三个指标才产生高于基线的信号。若无心智理论提示，中等能力模型的指标接近随机水平。大型模型在无显式心智理论提示时也表现出一些协调，但效果小于显式提示的情况。

### 协调幻觉

若无统计控制，演示中的"涌现协调"往往反映了：

- 提示工程预设了协调（系统提示中写着"协同工作"）。
- 观察者偏见（我们看到预期的模式）。
- 对成功运行的事后选择。

宣传"涌现协调"但无可测量信号的生产系统应被视为营销手段。需先测量再宣称。

### 最小化的心智理论感知代理

结构：

```
agent state:
  own_beliefs:    {facts the agent believes}
  other_models:   {other_agent_id -> {beliefs_the_agent_attributes_to_them}}
  actions_last_N: [history of others' actions]

observation update:
  - update own_beliefs from direct observation
  - update other_models[agent_id] from their action + prior beliefs

action selection:
  - enumerate candidate actions
  - for each, predict what each other agent will do next given their modeled beliefs
  - pick action that maximizes joint outcome under those predictions
```

`other_models`属性是心智理论状态。一阶心智理论只保留一层。二阶心智理论则添加`other_models[i][other_models_of_j]`——我认为代理i对代理j信念的看法。

### 长时程的危害

Li等人记录：上下文限制导致代理忘记哪个信念属于谁。幻觉会向其他代理模型添加虚假信念。两者都会产生"我以为他以为X"的错误，并随时间累积。

论文及2024-2026年后续研究记录的缓解措施：

- **在提示中显式包含心智理论状态。** 结构化格式：`{agent_id: belief_list}`。强制检索以保持身份-信念绑定。
- **更短的推理链。** 每轮减少心智理论更新次数以减少幻觉累积。
- **外部心智理论存储。** 在LLM上下文外维护模型；每轮仅注入相关部分。

### 心智理论在生产中失效的场景

- **对抗性环境。** 具有良好心智理论的代理更容易被操纵（你可以建模他们对你的建模，然后利用）。
- **异构团队。** 当模型不同时，适用于一个对手的心智理论模型无法泛化。
- **依赖事实的任务。** 心智理论关乎信念；如果正确性取决于事实，心智理论可能分散注意力。

### 可实际测量的协调

团队协调是真实而非提示修饰的三个实用信号：

1. **随时间推移的互补性。** 在多轮任务中，代理的行动是否覆盖不相交的子任务？
2. **预见性。** 代理A在T+1轮的行动是否依赖于对B在T+2轮行动的预测，且该预测正确？
3. **纠正能力。** 当A在T轮误读B的信念时，A是否在T+2轮前纠正？

这些可在记录的多代理系统中进行测量。它们是"协调"叙事的实质版本。

## 动手构建

`code/main.py`实现了：

- `ToMAgent`——跟踪自身信念和对其他代理的信念模型。
- 一个合作任务：三个代理必须从三个盒子中收集三个token；每个盒子只能放一个token。代理不能通信；它们通过彼此的行动推断意图。
- 两种配置：`zeroth_order`（无心智理论）和`first_order`（具有一级信念模型的心智理论）。
- 200次随机试验的测量：完成率、重复率（两个代理瞄准同一盒子）、平均完成轮次。

运行：

```
python3 code/main.py
```

预期输出：零阶代理的重复率约为35%，在10轮内完成约60%的试验。一阶心智理论代理的重复率约为5%，完成率约95%。这一差异即为可测量的协调效应。

## 交付检查

协调声明检查清单：

- **对照条件。** 移除协调提示的系统版本。同时测量两者。
- **统计检验。** 系统与对照在您的指标上是否在`p < 0.05`水平显著？
- **互补性测量。** 随时间的行动不相交性，而非仅最终成功。
- **故障案例记录。** 当代理协调失败时，心智理论状态是什么样子？
- **模型能力披露。** 若该效应在较小模型上消失，应明确说明。

4. 阅读Li等人（arXiv:2310.10701）。重现"长时程退化"发现：当轮次从10增加到30时，您的一阶心智理论性能如何变化？
5. 阅读Riedl 2025（arXiv:2510.05174）。在您的模拟日志上实现高阶协同效应统计量。在无心智理论提示条件下，该效应是否存在？

## 关键术语

| 术语 | 人们怎么说 | 实际含义 |
|------|----------------|------------------------|
| 心智理论 | "理解他人内心" | 建模其他代理信念的能力。按阶次分级（0、1、2+）。 |
| 萨利-安妮测试 | "错误信念测试" | 1985年发展心理学测试；LLM通过简单版本，在复杂版本中失败。 |
| 一阶心智理论 | "A相信X" | 建模另一个体对事实的信念。 |
| 二阶心智理论 | "A相信B相信X" | 深一层的递归建模。 |
| 与身份关联的分化 | "随时间保持稳定角色" | Riedl的指标：角色持续存在，而非随机。 |
| 目标导向的互补性 | "行动不相交" | 代理针对不同子任务，而非相同。 |
| 高阶协同效应 | "群体超越任何子集" | Riedl用于真实协调的统计度量。 |
| 协调幻觉 | "看起来协调" | 在无可测量信号的情况下，通过提示修饰表现出的协调外观。 |
