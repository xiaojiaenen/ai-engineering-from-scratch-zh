## 每课文档结构

打开 `docs/zh.md` 后通常包含：
- Motto / 引言
- 学习目标
- Problem / 问题
- Concept / 概念
- Build It / 构建它（手写代码部分）
- Use It / 使用它（工业实现对比）
- Ship It / 产出
- Exercises / 练习
- Key Terms / 关键术语
- Further Reading / 延伸阅读

## 标准学习流程

1. 读 Problem 和 Concept
2. 手敲 Build It 的代码（不要复制粘贴）
3. 跑通代码验证
4. 看 Use It 对比工业实现
5. 做练习（可选）
6. 标记完成
